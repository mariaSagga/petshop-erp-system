# PetShop ERP System

## Description

PetShop ERP System is a web-based management application developed for pet shop businesses.

The system helps manage:

- Customers
- Product inventory
- Sales
- Grooming appointments
- Stock updates
- Appointment scheduling

This project was developed using Flask and MySQL as part of a university assignment.

---

## Technologies Used

- Python
- Flask
- MySQL
- HTML5
- CSS3
- JavaScript

---

## Main Features

### Dashboard
- Business statistics overview
- Total customers
- Total products
- Total sales
- Grooming appointments
- Low stock alerts

### Customer Management
- Add customers
- Edit customer information
- Delete customers

### Inventory Management
- Add products
- Edit products
- Delete products
- Stock tracking
- Low stock warnings

### Sales Management
- Register product sales
- Automatic stock updates
- Product filtering

### Grooming System
- Grooming appointment scheduling
- Appointment calendar
- Prevention of duplicate appointments

### UI Features
- Responsive modern layout
- Dark mode
- Collapsible sidebar
- Search functionality
- Interactive interface using JavaScript

---

## Database Structure

Main tables used:

- users
- customers
- products
- orders
- order_items

---

## Installation

1. Install Python
2. Install Flask
3. Install MySQL
4. Import the database
5. Run:

```bash
python app.py
```

---

## Project Purpose

The purpose of this project is to simulate a real ERP management system for a pet shop business.
